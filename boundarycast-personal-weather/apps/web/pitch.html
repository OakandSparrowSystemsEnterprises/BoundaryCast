What we built

A governed weather oracle recipe

Anyone can create a weather-dependent market. Not everyone can write safe resolution rules. Our recipe packages the resolution logic: what evidence counts, when official alerts govern, when the claim is exact versus area-level, and when the oracle must abstain.

location → evidence → epistemology → claim scope → deontic policy → gatekeeper verdict → hash-chained artifact → replay

The key idea

Claim Scope is the resolution rule

exact_location → microclimate_adjusted → nearby_observation_area → official_forecast_area

  • The oracle states how specific its claim is; the market creator sets the minimum scope it will accept.
  • Evidence below the minimum never silently resolves — it goes UNRESOLVED → arbitration with an explicit reason code.
  • The oracle never pretends.
