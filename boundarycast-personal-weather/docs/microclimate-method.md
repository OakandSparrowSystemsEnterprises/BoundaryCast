# Microclimate Method v0.1

BoundaryCast treats microclimate evidence as bounded knowledge, not automatic certainty.

The first version captures location precision, surface exposure, shade exposure, elevation, wind exposure, water proximity, urban heat risk, observation distance, forecast grid distance, and sensor freshness.

Unknown values are valid. Unknowns reduce microclimate confidence. The system should never invent local certainty simply because the user asked for a personal forecast.
